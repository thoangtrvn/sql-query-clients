ibmcloudsql.utilities¶

The module provides IBMCloudAccess class, and some module-level utilities.

  • rename_keys(): rename keys in a dict

  • static_vars(): decorator that assign initial values to a ‘static’ variables in function, access to these variables require using also <func_name>.
